## Remaining gaps after this update

- Full live conformance harness coverage for interop scenarios is still pending (current additions are scenario definitions in the journey catalog).
- Deep payer-specific companion-guide validation for X12/NCPDP remains intentionally out-of-scope in this open repository (licensed specifications).
- Expand conformance harness execution to include these new negative-path journeys as first-class CI gates.
